ConstructTrack Fiber Optic Installation Management API

The ConstructTrack API provides comprehensive endpoints for managing fiber optic construction projects, field workers, tasks, and customer interactions. This API is designed specifically for the fiber optic installation industry with features like GPS tracking, photo documentation, and real-time progress updates.

Authentication

Most endpoints require authentication using JWT tokens. Include the token in the Authorization header:

Authorization: Bearer <your-jwt-token>
