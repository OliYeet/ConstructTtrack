## 🤝 Contributing to Documentation

We welcome contributions to improve our documentation:

1. **Found an error?** Create an issue or submit a PR
2. **Missing information?** Add it following our [documentation standards](development/documentation-standards.md)
3. **Unclear instructions?** Help us make them clearer

### Documentation Standards

- Use clear, concise language
- Include code examples where applicable
- Keep screenshots and diagrams up to date
- Follow the established file structure
- Update the status table when adding new documentation
